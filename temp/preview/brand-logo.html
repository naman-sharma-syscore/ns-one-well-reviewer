Usage rules

  • Don't recreate marks. Use the SVG files verbatim from assets/. Never redraw, retrace, or stylize.
  • Don't recolor seals. Brand seals carry official meaning — recoloring breaks the system. For dark surfaces, invert via CSS filter only.
  • Min sizing. Standard wordmark ≥ 24px tall. IWBI logo ≥ 32px. Watermarks ≥ 48px.
  • Clearspace. Reserve at least 0.5× the mark's height as padding around any logo.
  • One CTA gradient per view. The bright cyan gradient is loud — using it twice dilutes the primary action.
  • Performance gradient is reserved. Only on portfolio rating visualizations. Never as page background, never under body copy.
